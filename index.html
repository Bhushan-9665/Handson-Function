        Que- 4

        var x = 21;
            var girl = function () {
                console.log(x);
                var x = 20;
            };
            girl ();

    
        Variable hoisting means that, during the execution phase, all variable declarations with the var keyword 
        are moved to the top of their respective function scope, while their assignments remain in place. However, 
        the assignment of a value to a variable happens exactly where it appears in the code.
    
        Print output: 
            var x = 21;
            girl ();
            console.log(x)
            function girl() {
                console.log(x);
                var x = 20;
            };
    
        ans - 
        So, the final output will be undefined followed by 21. The reason for the undefined is the variable 
        hoisting inside the girl function, which makes the local variable x accessible but not yet assigned 
        a value at the time of the first console.log(x) inside the function. The global variable x is not 
        affected by the local variable inside the function and retains its original value of 21.

Que- 7 :- Write a function that accepts parameter n and returns factorial of n

                function factorial(n) {

                    if (n === 0 || n === 1) {
                        return 1;
                    } else {
                        return n * factorial(n - 1);
                    }
                }
                
                const number = 3;
                const result = factorial(number);
                console.log(result);
